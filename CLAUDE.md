# Fado Event Planning – Project Context

## What this project is
A website for **Fado Event Planning**, a wedding planning company. The name "Fado" is inspired by Portuguese soul music — the brand identity is built around deep emotion, timeless elegance, and heartfelt dedication.

## Brand identity
- **Tone:** Romantic, elegant, emotional, premium
- **Color palette:** Deep navy/purple (`#1a1a2e`, `#2d1b69`) with gold (`#c9a84c`, `#e8d5a3`) accents on a warm off-white (`#f9f6f0`) base
- **Typography:** Georgia serif for body, classic and refined
- **Feel:** Luxury wedding planner — think high-end stationery, candlelight, timeless not trendy

## Current state
- Single-page website (`index.html`) with: Nav, Hero, Services, Why Us (with stats), Gallery (placeholders), Testimonials, Contact form, Footer
- Gallery section uses CSS gradient placeholders — real photos to be added later
- Contact form is front-end only (no backend yet)

## Services offered
Full Wedding Planning, Floral & Décor Design, Catering & Menu Curation, Photography & Video, Entertainment & Music, Destination Weddings

## Key stats shown on site
- 250+ weddings planned
- 12 years experience
- 98% happy couples
- 30+ venue partners

## How this project was started
Created on 2026-05-21 as part of a multi-site setup (alongside Spark Digital Agency) inside the parent `Magda` folder. The owner is building websites for multiple clients and wants Claude to remember context across sessions.

## Ongoing preferences
- Keep all CSS inside the single `index.html` file (no separate stylesheets unless specifically requested)
- Maintain the existing color palette and brand identity for any new sections
- When adding real content (photos, team bios, etc.), the owner will provide assets — do not invent real business details like phone numbers or addresses unless asked
- GitHub repo: to be connected (owner will create repo and share URL)
